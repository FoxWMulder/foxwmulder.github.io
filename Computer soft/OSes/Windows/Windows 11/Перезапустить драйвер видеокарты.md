### Горячие клавиши перезапуска драйвера видеокарты  
  
**Windows + Ctrl + Shift + B** — сочетание клавиш для быстрого перезапуска драйвера видеокарты, предусмотренное в Windows 11 и Windows 10.  
  
### Вы можете использовать PowerShell или Терминал Windows  
  
Cледующие две команды:  
  
`pnputil /enum-devices /class Display`  
`pnputil /restart-device "полученный ID"`  
  
в первой получаем нужный ID устройства, во второй — выполняем сброс, указав ранее полученный ID.![Сброс драйвера видеокарты в PowerShell](attachments/Pasted%20image-2025-11-19%2002-40-06-598.png)
