```ad-desc  
Возвращает либо домашний путь пользователя (в Linux - /home/<username>), либо путь к временной папке приложения (В Windows Vista и позже - C:\\Users\\<username>\\AppData\\Roaming), либо путь к хранилищу (в Android - root/data/data/<application ID>/files).  
  
```
